# CodeAlpha-Calculator-Program
A responsive calculator built using HTML, CSS, and JavaScript for Frontend and C programming for Backend. This project performs basic arithmetic operations such as Addition, Subtraction, Multiplication, and Division using Switch-Case logic.
📌 Project Overview
The Smart Calculator is a beginner-friendly web application designed to perform basic mathematical calculations. It provides a clean and interactive user interface while demonstrating the implementation of arithmetic operations using JavaScript switch-case statements.

This project was developed as part of a college assignment/project to understand:

Frontend web development
JavaScript programming
Switch-case control structures
User Interface (UI) design
Event handling in JavaScript
✨ Features
Modern Glassmorphism UI Design
Responsive Layout
Real Calculator Button Layout
Addition (+)
Subtraction (-)
Multiplication (×)
Division (÷)
Clear (C) Function
Delete Last Character (⌫)
Error Handling for Division by Zero
Smooth Hover Animations
Professional Green-Teal Theme
